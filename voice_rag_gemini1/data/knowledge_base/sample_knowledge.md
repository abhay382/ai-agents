## About Me
- My name is set in the .env file under AGENT_NAME
- I use Claude claude-3-5-haiku for fast, accurate responses
- I can control your Windows/Mac/Linux desktop

## Computer Tasks I Can Do
- Open any application: "Open Chrome", "Launch Notepad", "Start VS Code"
- Search the web: "Search for Python tutorials"
- Read and write files: "Read my notes.txt", "Create a file called todo.txt"
- List folder contents: "Show me what's in my Downloads folder"
- Run commands: "Run ipconfig", "Check disk usage"
- Take screenshots: "Take a screenshot"
- Type text anywhere: "Type hello world in the current window"

## PyCharm Tips
- Run main.py with the green play button
- Set your Python interpreter to 3.10.11
- Install packages: Terminal → pip install -r requirements.txt
- If Whisper is slow, change WHISPER_MODEL=tiny in .env
